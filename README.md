# Tractor Supply Stall Mat Price Comparison

Interactive map viewer showing zone-based pricing for Tractor Supply stall mats across ~2,289 US store locations.

**[View Live Site](https://ctenold.github.io/Tractor-Supply-Prices/)**

## Features

- **Product Selection**: Browse 6 rubber mat products in the sidebar
- **Interactive Map**: Leaflet-powered map with color-coded price zones
- **Two View Modes**:
  - **By Product**: See all store prices for a selected product on the map
  - **By Location**: Select a store to see all product prices at that location
- **Data Tables**: Searchable, sortable tables with CSV export (powered by DataTables)
- **Zone Summary**: View price zones with store counts and pricing tiers

## Data

- **2,289 stores** with GPS coordinates across the continental US
- **Zone-based pricing** showing regional price variations
- Currently showing actual price data for the 4x6x3/4" Thick Rubber Stall Mat (SKU 221900399)

## Tech Stack

- Leaflet.js for interactive mapping
- DataTables for sortable/searchable tables
- jQuery
- All dependencies loaded via CDN (no build step required)

## Local Development

Simply open `index.html` in a browser. No server required.
